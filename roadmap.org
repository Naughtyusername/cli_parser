* Side Projects (use this library)
Build real tools to dog-food the library. In order of complexity:

** TODO 1. wc clone (word/line counter)
Simple, quick win. Learn file I/O basics.
- Positional: files to count
- Flags: -l (lines), -w (words), -c (chars)
- Bonus: -v (verbose), multiple files support

** TODO 2. grep-like search tool
Pattern searching in files.
- Option: --pattern/-p for search term
- Positional: files to search
- Flags: -i (case insensitive), -v (invert), -n (line numbers)

** TODO 3. File renamer
Batch rename utility.
- Options: --find and --replace patterns
- Positional: files to rename
- Flags: --dry-run, -v (verbose)

** TODO 4. Todo manager
Simple task tracking, stepping stone to subcommands.
- Positional commands: add, list, done, remove
- File I/O for persistence
- Flags: --all, --completed
